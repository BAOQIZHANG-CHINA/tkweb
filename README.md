# 迪拜旅游网站

这是一个响应式的迪拜旅游信息网站，展示迪拜的景点、美食、购物和旅行贴士。

## 设计特点

- **主色调**: 金色 (#FFD700)、深蓝色 (#0A2463)、沙色 (#F5DEB3)、白色
- **字体**: 标题使用阿拉伯风格字体 (Tajawal)，正文使用无衬线字体 (Poppins)
- **整体风格**: 奢华、现代、中东风情融合
- **背景元素**: 微妙的阿拉伯几何图案

## 功能特性

1. **响应式设计**: 适配手机、平板、桌面设备
2. **导航栏**: 固定顶部，半透明背景，滚动时变实色
3. **景点模块**: 卡片式布局，悬停效果
4. **美食推荐**: 展示阿拉伯特色美食
5. **购物指南**: 介绍迪拜购物中心、黄金市场、香料市场
6. **旅行贴士**: 签证、礼仪、交通等实用信息
7. **交互功能**:
   - 天气小部件（显示迪拜实时天气）
   - 货币转换器（AED ↔️ CNY/USD）
   - 行程规划器（用户可收藏景点生成行程单）
   - 搜索功能

## 文件结构

```
├── index.html          # 主页面
├── styles.css          # 样式文件
├── script.js           # JavaScript功能
└── README.md           # 项目说明
```

## 部署说明

要预览此网站，只需在本地服务器上打开 `index.html` 文件，或将其部署到任何支持静态文件的Web服务器上。

## 技术栈

- HTML5
- CSS3 (含响应式设计、动画效果)
- JavaScript (ES6+)
- Google Fonts (Tajawal, Poppins)
